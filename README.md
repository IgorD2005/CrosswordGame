This repository contains a Python script that allows users to play a crossword game. The game is designed for 1 to 4 players and follows good programming standards and practices. The project includes automatically generated documentation in PDF format.

Features
1. Graphical User Interface (GUI) - user-friendly interface for the game.
2. Player Registration and Login Module - supports multiple players.
3. Game Settings Module - allows customization of game options.
4. Player Database (ORM) - stores player information securely.
5. Questions and Answers Database - contains all crossword questions and answers.
6. Two Game Modes - for example, timed mode and “first mistake” mode.
7. Score Display Module - shows player results during and after the game.
8. Export Results to PDF - allows saving game results in a readable format.
9. Encrypted Player Data - all sensitive player data is stored securely.
