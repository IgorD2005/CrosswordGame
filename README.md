# Crossword Game

This repository contains a Python script that allows users to play a crossword game. The game is designed for **1 to 4 players** and follows good programming standards and practices. The project also includes automatically generated documentation in **PDF** format.

---

## Features

- 🖥️ **Graphical User Interface (GUI)** – user-friendly interface for the game.
- 👤 **Player Registration and Login Module** – supports multiple players.
- ⚙️ **Game Settings Module** – allows customization of game options.
- 🗄️ **Player Database (ORM)** – stores player information securely.
- ❓ **Questions and Answers Database** – contains all crossword questions and answers.
- 🎮 **Two Game Modes** – for example, timed mode and *First Mistake* mode.
- 🏆 **Score Display Module** – shows player results during and after the game.
- 📄 **Export Results to PDF** – allows saving game results in a readable format.
- 🔒 **Password Hashing** – player passwords are hashed using **SHA-256** before storage.
